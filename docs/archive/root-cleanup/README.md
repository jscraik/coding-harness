## Authority

This archive is evidence, not an active planning or policy surface. Use current
plans, specs, Linear issues, and governance docs for implementation authority.
Archived evidence subdirectories are excluded from the changed-doc Vale pass so
historical records do not need prose rewrites to satisfy current vocabulary.
Archive-internal Markdown references are guarded by
`pnpm run docs:root-archive:links`.
