PU-013 admits codex-runtime-evidence/v1 packets at the runtime-card CLI boundary, normalizes them through the existing runtime evidence bundle seam, and projects compact runtime truth into runtime-card/v1 plus harness next --json. The artifact trust boundary is shared by runtime-card --evidence, harness next --runtime-card, and the sibling --phase-exit path so absolute, traversal, and symlink-escaping runtime artifacts fail closed inside --repo. The first adversarial implementation review found a TOCTOU risk between path validation and file read; the fix now opens the canonical artifact with O_NOFOLLOW, verifies the opened descriptor with fstat, reads from that descriptor, and closes it in finally.
